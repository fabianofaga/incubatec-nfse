Começe rápido

É simples e de fácil de configurar.

Introdução Instalação Cadastros Contribuição
Para utilizar esse sistema você precisa necessariamente de um software que simule um sevidor web respectivamente com as versões descritas nesse documento. Atualmente existe diversas opções para esse tipo de software, caso você seja um usuário que não possui conhecimentos prévios sobre esse assunto entre em contato ou procure por um profissional com conhecimentos necessários. Dentro as diversas opções indicamos o software WampServer, o qual é bem simples e possui varios contéudos na Internet. Os arquivos gerados(Identificação, Arquivo Mestre, Destinatário e Item de documento) por esse sistema serão utilizados no Validador que pode ser encontrado através clicando aqui.

Requisitos:

PHP: 5.6 ou superior

Apache: 2.4

MySQL: 5.7

Baixar/Download

Faça o download ou clone diretamente do repositório. Importe o sql do banco de dados, edite a classe responsável pelo banco de dados( Database.php ).

Bibliotecas utilizadas

Bootstrap

jQuery Confirm

jQuery Mask Plugin

Data Tables

Primeiros passos

Após realizar o download, abra o arquivo Database.php e edite as variaveis para conexão com o banco.
  1. IncubaNFSe
  2. sys
  3. Database.php

private $server = 'localhost';IP do servidor db

private $user = 'root'; Usuário do db

private $password = ' '; Senha do usuário

private $db = 'incubatec_nf'; Nome do banco

Importe o banco de dados presente na raiz do repositório database-atualizado.sql.
