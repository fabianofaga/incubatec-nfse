Introdução Configurar DB Como usar Contribuição
Para utilizar esse sistema você precisa necessariamente de um software que simule um sevidor web respectivamente com as versões descritas nesse documento. Atualmente existe diversas opções para esse tipo de software, caso você seja um usuário que não possui conhecimentos prévios sobre esse assunto entre em contato ou procure por um profissional com conhecimentos necessários. Dentro as diversas opções indicamos o software WampServer, o qual é bem simples de facil instalação e contém todos os softwares necessários para um servidor web. Os arquivos gerados(Identificação, Arquivo Mestre, Destinatário e Item de documento) por esse sistema serão utilizados no Validador que pode ser encontrado através clicando aqui.

Requisitos:

PHP: 5.6 ou superior

Apache: 2.4

MySQL: 5.7

Baixar/Download

Faça o download ou clone diretamente do repositório. Importe o sql do banco de dados, edite a classe responsável pelo banco de dados( Database.php ).

Primeiros passos

Após realizar o download, abra o arquivo Database.php e edite as variaveis para conexão com o banco.
  1. IncubaNFSe
  2. sys
  3. Database.php

private $server = 'localhost';IP do servidor db

private $user = 'root'; Usuário do db

private $password = ' '; Senha do usuário

private $db = 'incubatec_nf'; Nome do banco

Importe o banco de dados presente na raiz do repositório database-atualizado.sql.

Cadastros em gerais*

Esse sistema(ainda) não possui todos os tipos de validações e verificações nos campos de preenchimento pelo usuário. Portanto é indicado que o usuário preencha completamente todos os campos conforme descrito nas observações.

Convênio ICMS 115/03

Para evitar qualquer problema relacionado ao preenchimento dos dados, ler convênio ICMS 115/03

Tudo pronto e configurado

Após instalação e importação do banco de dados devidamente concluída é hora de realizar o cadastro da empresa, o mesmo será realizado apenas uma vez e caso necessite alterar os dados basta voltar a tela de cadastro clicar no botão "alterar" e logo em seguida salvar. Há também uma tela para cadastrar os clientes, com campos específicos para geração dos arquivos finais.

E por fim, a tela de nota fiscal, na qual deve-se tomar bastante atenção em seu presente preenchimento. O cliente é localizado pelo seu CPF/CNPJ cadastrado no sistema.

Cadastrei tudo, e agora?

Agora é hora de fazer as suas notas! Clique no menu 'Nova nota Fiscal', preencha corretamente todos os campos e conclua. Com isso feito é hora de gerar remessa de arquivos. Clique no menu 'Remessa', escolha o mês e modelo clique sobre o botão gerar e aguarde até o download do NFSe.zip.

Arquvios extraídos do ZIP

Arquivo mestre, destinatário e item.

A partir de agora você deve baixar o Valida e-Nota Fiscal e carregar o arquivo de identificação. Após carregado, uma nova janela onde você irá carregar o arquivo Mestre e caso tudo esteja preenchido corretamente e sem erros, será gerado mais dois arquivos, Y e C. (O arquivo de log sempre vai ser criado mesmo que não exista erros).

Depois de gerado todos os arquivos pelo programa valida, será utilizado o programa Gera Mídia TED - Versão 3.00d que converterá os arquivos de texto em arquivos Ted para ser enviados ao fisco pelo programa TED.
