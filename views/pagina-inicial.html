Incuba NFSe v1.0

Incuba NFSe é um projeto Open Source(Código Aberto) livre para baixar/utilizar e modificar. Criado para emissão de: Nota Fiscal de Serviço de Telecomunicações(modelo 22) e Nota Fiscal de Serviço de Comunicações(modelo 21). Regulamentado e embasado no CONVÊNIO ICMS 115/03 para mais informações sobre o convênio clique aqui.

De acordo com pesquisas realizadas pela Incubatec, foi constatado a falta de um software gratuito para a emissão de notas fiscais eletrônicas dos serviços de Comunicação (modelo 21) e de Telecomunicação (modelo 22). Por isso a Incubatec decidiu incubar esse projeto, visando facilitar a emissão para as pequenas empresas. O sistema foi desenvolvido com base nas normas do Convênio ICMS 115/03, publicado no DOU de 17.12.03.

Para começar a utilizar o sistema siga as instruções no menu instruções.
